# Checklist del repositorio público

| Elemento | Estado esperado |
|---|---|
| Nombre del repositorio | `universidad-programacion-movil-finmate` |
| Visibilidad | Público para revisión académica |
| README | Explica proyecto, alcance, estructura y ejecución |
| Documentación | Incluye documento de diseño y checklist de rúbrica |
| Código base | Incluye estructura inicial Ionic/React/Capacitor |
| Evidencias | Incluye referencias a mockups, flujo y validaciones |

## Validación sugerida después de publicar

```bash
git status
git log --oneline --decorate -5
```

## Capturas recomendadas

1. Página principal del repositorio.
2. README visible.
3. Carpeta `docs`.
4. Carpeta `src`.
5. Historial de commits.
