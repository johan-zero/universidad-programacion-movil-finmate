# Guion de demostracion

Duracion sugerida: 2 a 3 minutos.

1. Presentar FinMate Mobile y su arquitectura local.
2. Mostrar el inicio de sesion local.
3. Mostrar el Dashboard con ingresos, gastos y balance.
4. Registrar un gasto desde Nuevo movimiento.
5. Mostrar el historial actualizado.
6. Mostrar metas y porcentaje de avance.
7. Mostrar alertas y ajustes.
8. Mostrar el repositorio y diagramas UML.
9. Cerrar con la preparacion del proyecto Android.
