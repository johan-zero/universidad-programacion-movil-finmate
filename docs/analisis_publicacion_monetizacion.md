# Analisis de publicacion y monetizacion

## Publicacion

La primera plataforma objetivo es Android. Antes de publicar se deben preparar la ficha de tienda, icono, capturas, politica de privacidad y firma de la version de produccion. Para iOS se requiere cuenta Apple Developer, macOS con Xcode y App Store Connect.

## Privacidad

La version academica conserva los movimientos en el dispositivo y no usa un servidor externo. La politica de privacidad debe explicar que datos se almacenan localmente, como se eliminan y que no se comparten con terceros.

## Monetizacion

Se propone un modelo freemium sin publicidad invasiva. La version gratuita incluye registro, balance, metas y alertas. Una futura version premium podria incorporar exportaciones, categorias ilimitadas, copias de seguridad cifradas e informes.
